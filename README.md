
        * A simple Java Swing Alarm Clock application.
        * 
        * Features:
        * 1. Displays current time (HH:mm:ss).
        * 2. Allows user to set an alarm time (HH:mm).
        * 3. Shows a popup when the alarm time is reached.
        * Note: This is a basic implementation and can be enhanced with additional features
        * like multiple alarms, sound notifications, etc.
